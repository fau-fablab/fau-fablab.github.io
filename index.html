FAU FabLab GitHub

Diese Seite ist eine Überblicksseite über die Repositories, die für das FAU FabLab wichtig sind und auf GitHub liegen. Die Website des FAU FabLab befindet sich unter fablab.fau.de.

  • Ultraschallbad
  • Platinenätzer und -Fertigung
  • Reflow Ofen

  • OpenERP/Odoo Anleitungen
